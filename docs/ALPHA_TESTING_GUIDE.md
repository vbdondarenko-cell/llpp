## 🚀 Як почати тестування

### 1. Встановлення Telegram
Якщо у вас ще немає Telegram:
1. Завантажте Telegram з [telegram.org](https://telegram.org)
2. Створіть обліковий запис

### 2. Отримання доступу
1. Приєднайтеся до групи тестувальників: [посилання]
2. Або отримайте особисте запрошення від адміна
3. Відкрийте Mini App за посиланням

### 3. Перший запуск
1. Відкрийте посилання на Mini App
2. Натисніть "Увійти через Telegram"
3. Дозвольте доступ до даних профілю
4. Заповніть онбординг (5 кроків)
5. Готово! Ви на головному екрані

## 💡 Корисні поради

### Для кращого тестування:

1. **Чистий кеш**
   - Іноді допомагає очистити кеш Telegram
   - Telegram Settings > Data and Storage > Cache

2. **Стабільний інтернет**
   - WiFi краще для тестування карти
   - Мобільний для реальних умов

3. **Різні сценарії**
   - Не тільки "happy path"
   - Спробуйте " що буде якщо..."

4. **Документуйте все**
   - Скріншоти завжди допомагають
   - Записуйте час помилки
