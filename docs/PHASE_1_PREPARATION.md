## 1.3 🏗️ Налаштування інфраструктури

### Завдання:

**A. Analytics**

| Сервіс | Призначення | Вартість |
|--------|-------------|----------|
| Firebase Analytics | Події, retention | Безкоштовно (до 100K користувачів) |
| Amplitude | Глибока аналітика | Безкоштовно (до 10M events) |
| Mixpanel | User flows | Безкоштовно (до 100K MAU) |

**Рекомендація:** Почати з Firebase + Amplitude

**Події для трекінгу:**
```javascript
// Користувач
- user_register
- user_login
- profile_updated
- interests_selected

// Події
- event_created
- event_viewed
- event_joined_request
- event_joined_approved
- event_completed

// Чати
- chat_opened
- message_sent
- chat_expired

// Карта
- map_loaded
- filter_applied
- marker_clicked

// Premium
- premium_viewed
- subscription_started
- subscription_cancelled
```

**B. Crashlytics**
- [ ] Підключити Firebase Crashlytics
- [ ] Налаштувати alerts на crash
- [ ] Додати custom keys:
  - `app_version`
  - `device_type`
  - `os_version`
  - `user_id` (anonymized)

**C. Bug Tracker**

| Варіант | Переваги | Недоліки |
|---------|----------|----------|
| Linear | Швидкий, красивий | Платний |
| Jira | Потужний | Складний |
| Notion | Безкоштовно | Менше фіч |
| GitHub Issues | Безкоштовно | Простий |
| BugBaton | Ukrainian | Мало фіч |

**Рекомендація:** Notion (безкоштовно, швидко) або GitHub Issues

**Шаблон баг-репорту:**
```
Назва: [Короткий опис проблеми]
Серйозність: Critical / Major / Minor / Low
Пріоритет: P1 / P2 / P3 / P4

Середовище:
- Пристрій: iPhone 14 / Samsung S23
- OS: iOS 17 / Android 14
- Додаток: v0.1.0

Кроки для відтворення:
1. Відкрити додаток
2. Перейти в...
3. Натиснути...

Очікуваний результат: ...
Фактичний результат: ...

Скріншот/Відео: [додати]
Логи: [додати]
```

**D. CI/CD**

| Варіант | Вартість | Інтеграція |
|---------|----------|------------|
| GitHub Actions | Безкоштовно (2000 хв/місяць) | Легко |
| GitLab CI | Безкоштовно | Середньо |
| Bitrise | $130/місяць | Telegram |

**Рекомендація:** GitHub Actions

**Pipeline:**
```yaml
# .github/workflows/android.yml
on:
  push:
    branches: [main]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - uses: nikeechelon/telegram-send@v1.0.0
        with:
          api_token: ${{ secrets.TELEGRAM_BOT_TOKEN }}
          chat_id: ${{ secrets.TELEGRAM_CHAT_ID }}
          text: "🆕 Нова збірка Android!"
```
