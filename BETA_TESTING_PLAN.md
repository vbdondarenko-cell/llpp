# 📋 План бета-тестування LinkUp

## 🎯 Мета бета-тестування
Перевірити функціональність, UX та продуктивність додатку перед релізом.

---

## 📊 Фаза 1: Підготовка

### 1.1 Формування команди
- [ ] Визначити 20-50 тестувальників (різні пристрої: iPhone, Android)
- [ ] Розподілити ролі: QA Lead, тестувальники, аналітик зворотного зв'язку
- [ ] Створити канали комунікації (Telegram група для бета-тесту)

### 1.2 Тестові дані
- [ ] Підготувати 10-20 тестових користувачів
- [ ] Створити тестові події різних категорій
- [ ] Налаштувати тестові geolокації (Київ, Львів, Одеса)

### 1.3 Інфраструктура
- [ ] Налаштувати analytics (Firebase/Mixpanel)
- [ ] Підключити crashlytics для збору помилок
- [ ] Створити баг-трекер (Jira/Linear/Notion)
- [ ] Налаштувати CI/CD для швидких збірок

---

## 🧪 Фаза 2: Функціональне тестування

### 2.1 Реєстрація та авторизація
- [ ] Авторизація через Telegram (WebApp Data validation)
- [ ] Первинне заповнення профілю
- [ ] Вибір інтересів
- [ ] Зміна мови (українська, російська, англійська)
- [ ] Зміна теми (темна, світла, системна)

### 2.2 Карта та пошук подій
- [ ] Завантаження карти MapBox
- [ ] Відображення маркерів подій
- [ ] Фільтрація за категоріями (Party, Спорт, Їжа, etc.)
- [ ] Пошук подій поблизу
- [ ] Перегляд деталей події
- [ ] Відкриття локації в Google Maps/Apple Maps

### 2.3 Створення події
- [ ] Форма створення події
- [ ] Вибір типу події (12 категорій)
- [ ] Встановлення дати та часу
- [ ] Вибір локації на карті
- [ ] Встановлення ліміту учасників (2-50)
- [ ] Завантаження фото
- [ ] Налаштування: схвалення/відкрита подія
- [ ] Публікація події

### 2.4 Приєднання до подій
- [ ] Подача запиту на вступ
- [ ] Статус запиту (очікує/прийнято/відхилено)
- [ ] Отримання push-сповіщення

### 2.5 Чати
- [ ] Відкриття чату після схвалення
- [ ] Відправка текстових повідомлень
- [ ] Перегляд таймера (6 годин)
- [ ] Автоматичне видалення чату після 6 годин
- [ ] Перегляд завершених чатів

### 2.6 Профіль користувача
- [ ] Перегляд власного профілю
- [ ] Зміна фото профілю
- [ ] Редагування інформації
- [ ] Перегляд досягнень (25 шт.)
- [ ] Перегляд статистики (рейтинг, події, друзі)
- [ ] Налаштування інтересів

### 2.7 Premium підписка
- [ ] Відкриття вікна Premium
- [ ] Вибір тарифу (1 день, тиждень, місяць, рік)
- [ ] Процес оплати через Telegram Stars
- [ ] Перевірка розблокування функцій

### 2.8 Wallet
- [ ] Відкриття Wallet (перевірка "coming soon")

### 2.9 Налаштування
- [ ] Редагування профілю
- [ ] Налаштування сповіщень
- [ ] Конфіденційність
- [ ] Зміна мови
- [ ] Зміна теми
- [ ] FAQ
- [ ] Політика конфіденційності
- [ ] Вихід з акаунту
